### Геномный браузер (GeMLBee)

yaml config help:
```yaml
genome: <UCSC genome>
tracks:
    - path/to/datafile
```

yaml config example:
```yaml
genome: hg19[chr1, chr2]
tracks:
    - path/to/track.extention
```
